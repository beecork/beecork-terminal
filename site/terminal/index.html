What's inside

Terminal first. Everything around it.

The terminal is the star

A real, GPU-rendered terminal running your login shell — launch claude or any CLI agent and work full-size.

Live diff view

As the agent edits, the file tree colors changed folders and files, and each file shows a line-level diff against git.

File browser + editor

An expandable side panel: browse the folder, open files, edit with syntax highlighting, split into two views.

Sessions that stay alive

A left rail of terminal sessions — each keeps running in the background. Rename them, pin the rail, ⌘T for a new one.

Clickable & searchable

Click a file:line in output to open it. ⌘F to search scrollback. Alerts when a background agent needs you.

Lean & themeable

Built with Tauri — a small, fast native app. Multiple color themes, adjustable font, tuned for how you work.
